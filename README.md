# Super Volleyball
[![MIT](https://img.shields.io/github/license/Namr/SuperVolleyball)](https://github.com/namr/supervolleyball/blob/main/LICENSE-MIT)

An online multiplayer clone of the [best Mario Party minigame](https://www.mariowiki.com/Beach_Volley_Folly). Runs in the browser via WebAssembly. Work in progress, a link to a hosted version will be added here once ready.
